Erie County Food Facility Inspections

This site is a demonstration of one use of government data when it is opened up for public use. Find this project and others at Buffalo Open Data.

What is Open Data?

There are many definitions of "Open Data" but, at a minimum, it can be thought of as data that can be freely used, reused and redistributed by anyone.

Where did this data come from?

The inspection data on this site was obtained from the Erie County (NY) health inspection database. The data was then augmented with geocodes (latidudes/longitudes) by interpolating the coordinates from US Census GIS files. The address interpolation is not 100% accurate so there may be some points that appear in the wrong place on the map.

What is the point of this site?

This site serves as just one example of what can be done when data is made accessible to the general public. The more data sets that are made available, the richer the applications can become as data is "mashed-up" in novel ways. These uses can run the gamut from simple mobile/web apps to in-depth analysis.

What powers the map?

The map on this site uses the OpenLayers library for display and the data comes from a custom developed API

Where is the source code?

This site and the tools used to create it are Open Source and the source code can be found on GitHub.
